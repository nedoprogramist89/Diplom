## 4. Оформление (разд. 4 методички)

Проверка **вручную при финальной вёрстке в Word**:

- Поля: слева **3,5** см, сверху **2,5**, справа **2**, снизу **2** см; абзац **1,25** см.
- Шрифт основного текста **14** pt, **1,5** интервал; таблицы и приложения **12** pt (п. 4.1).
- Подписи рисунков: **«Рисунок N – …»** под рисунком, шрифт **12** pt (п. 4.5).
- Таблицы: **«Таблица N – …»** над таблицей слева (п. 4.6); на каждую таблицу — **ссылка** в тексте основной ПЗ.
- Нумерация страниц сквозная, колонтитул **одна строка** (п. 4.1).
- В тексте ПЗ не использовать **№**, **%**, **=** как одиночные символы вместо слов (п. 2.2 методички) — в технических приложениях к URL и коду допустимы оговорки; в **пояснительной записке** вычитать отдельно.

Текстовые файлы в `docs/` без колонтитулов — это **исходники**; итоговая проверка — после экспорта в **.docx**.

1. Заменить в «Документ 4» в содержании номера страниц на **фактические** после вёрстки.

5. Подшивка, файлы для отзыва/рецензии/антиплагиата — по п. **4.2**, **6.6**.
